# ultrafastparrot cult
This is the webring for the ultrafastparrot cult, made with astro, and hosted on cloudflare workers.

## Joining
Fork and open a PR adding a site to `sites.ts`

Also update your site to include something like
```
<a href="https://ultrafastparrot.net/prev/YOUR_SLUG">Previous</a>
<a href="https://ultrafastparrot.net/">ultrafastparrot</a>
<a href="https://ultrafastparrot.net/next/YOUR_SLUG">Next</a>
```
